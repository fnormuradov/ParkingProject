Есть парковка для машин. В ней есть несколько мест для легковых автомобилей и несколько для грузовых.
Легковая, при отсутствие места для легковых машин, может занять место для грузовой машины. Грузовая машина может занять место только для грузовых.
На въезде в парковку, если есть свободные места, машина получает талон с номером места. На выезде его отдает.
Реализовать метод "парковки", который вернет номер места для автомобиля и метод "выезда из парковки" по номеру места.

На выходе ожидается рабочий код.
Если код будет покрыт тестами это большой плюс.
